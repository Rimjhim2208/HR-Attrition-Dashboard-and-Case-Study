# HR Attrition Analysis Dashboard & Employee Retention Case Study

## Project Overview

This project analyzes employee attrition using the IBM HR Analytics dataset and presents insights through an interactive Power BI dashboard and a business case study.

The objective was to identify workforce turnover patterns, understand key attrition drivers, and provide data-driven recommendations to improve employee retention.

## Key Findings

* Overall attrition rate is approximately 16.1%.
* Employees working overtime show higher attrition.
* Lower salary bands experience higher turnover.
* Job satisfaction and work-life balance significantly influence retention.
* Promotion delays are associated with increased attrition risk.
* Certain departments and job roles contribute disproportionately to employee exits.
